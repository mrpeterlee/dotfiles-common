# Git Workflow Prompt

Branch naming: `<prefix>/<description>` where prefix is one of: feature/, bug/, fix/, hotfix/, chore/, docs/, test/, refactor/

Commit messages: `<type>: <description>` in lowercase imperative mood, under 72 chars.
Types: feat, fix, docs, chore, test, refactor, ci

Every PR must include: purpose, risk assessment, verification evidence.
